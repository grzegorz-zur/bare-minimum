# Bare Minimum

Minimalistic text editor for GNU/Linux.

## Keys

### Normal mode

* space — insert mode
* delete — delete character
* arrows — move

* d — left
* f — right
* j — down
* k — up

* ^q — quit editor
* ^w — close current file
* ^e — write current file
* ^z — stop the editor (background)
* ^d — switch to previous file
* ^f — switch to next file

### Input mode

* escape — normal mode
* enter — split line
* delete — delete character
* arrows — move
